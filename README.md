# AI Anime Short Drama Tech Plan

This repository stores the technical plan, reusable templates, and version history for an AI anime short drama production workflow.

## Repository Structure

- `ai-anime-short-drama-tech-plan.md`: full technical plan in Chinese
- `templates/csv/scene_list.csv`: scene planning template
- `templates/csv/shot_list.csv`: shot planning template
- `templates/prompts/`: reusable prompts for `Claude`, `Gemini`, and `GPT`
- `checklists/delivery-checklist.md`: production and delivery checklist
- `docs/plans/`: repository design and planning notes

## Purpose

- Version-manage the production methodology
- Reuse planning and prompting templates across projects
- Keep pre-production artifacts consistent and reviewable
- Provide a clean starting point for future AI short drama projects
