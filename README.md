# AI 动漫短剧技术方案仓库

这个仓库用于保存 AI 动漫短剧生产流程的技术方案、可复用模板，以及后续版本演进记录。

## 仓库结构

- `ai-anime-short-drama-tech-plan.md`：完整技术方案正文
- `templates/csv/scene_list.csv`：场次规划模板
- `templates/csv/shot_list.csv`：镜头规划模板
- `templates/prompts/`：`Claude`、`Gemini`、`GPT` 的提示词模板
- `checklists/delivery-checklist.md`：生产与交付检查清单
- `docs/plans/`：仓库设计与规划说明

## 用途

- 对 AI 动漫短剧生产方法进行版本管理
- 在不同项目之间复用规划模板和提示词模板
- 保持前期策划资料结构稳定、便于审阅
- 作为后续 AI 短剧项目的标准起点

## 说明

- 仓库中的说明文档默认使用中文维护
- `CSV` 字段名保留英文，便于表格处理、脚本读取和跨工具兼容
