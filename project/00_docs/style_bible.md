# Style Bible

状态：demo draft，待确认

## 目标风格

- Anime suspense
- 真实校园比例，不走 Q 版
- 情绪优先于动作炫技

## 线条与上色

- 线条：中细线，轮廓清楚，五官不过度锐利
- 上色：赛璐璐为主，局部柔和阴影辅助
- 光影逻辑：室内冷白光，走廊与黄昏偏冷橙混合

## 镜头语言

- 多用中近景、反应镜头和插入镜头
- 悬疑信息优先通过视线、停顿和构图遮挡传递
- 避免连续长镜头和过度复杂运动

## 色板

- 校园室内：灰蓝、米白、木色
- 紧张提示物：偏冷青或高对比白纸
- 危险存在感：深灰、暗青、低饱和阴影

## 背景复杂度

- 教室和走廊保持中等复杂度
- 不要堆满装饰物
- 背景要服务角色站位和信息读取

## 人物与场景融合

- 人物边缘不能从背景里糊掉
- 情绪镜头优先保证面部和手部可读性
- 剪影镜头允许背景比人物更亮，但主体轮廓必须清晰

## 禁止元素

- 霓虹赛博风
- 过亮糖果色
- 夸张搞笑表情
- 明显模仿已知商业 IP 角色造型
